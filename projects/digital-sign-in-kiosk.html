Digital Sign-In & Locker Kiosk

Industrial Cadets Gold Project (2022/23) — A smart, modular safety kiosk featuring automatic locker locking, LED occupancy status, and an interactive site-hazard map.

ROLE
Mechatronics & Systems
TIMELINE
2022 / 2023
STACK
C++ · Python · Fusion 360
OUTCOME
3D Printed Operational Prototype
01 — THE BRIEF

Designed a mechatronic sign-in station to manage safety regulations on Severn Trent Water facilities. The kiosk provides visual safety guidance, safe storage for potential ignition hazards (like phones/lighters) via automatically locked compartments, and logs worker presence.

View Full Report PDF ↗
02 — TECHNICAL DEPTH & ACHIEVEMENTS
  • Firmware Development: Developed C++ firmware on an Arduino Mega 2560 to manage locker servos, RGB status LEDs, and LDR light sensing. Programmed the Serial parsing logic to decode incoming instructions dynamically.
  • Kivy-Arduino Serial Integration: Integrated Kivy-based Python GUI on a Raspberry Pi with the Arduino microcontroller over USB using PySerial. Handled latency constraints and optimised data transmission using short, robust integer instruction tokens (e.g. 10/11 for lock/unlock).
  • Modular CAD Design: CAD modelled the modular stackable lockers in Fusion 360. Incorporated slots/grooves for structural stacking, dedicated cavities for servo and LED integration, and tolerance cutouts for USB charging ports.
SKILLS USED: C++ Python Fusion 360
03 — OUTCOME

Successfully built and verified a mechatronic kiosk prototype with automatic locker locks, real-time occupancy status indicators, and an interactive safety map. Recognised for successful cross-disciplinary hardware-software systems integration.
