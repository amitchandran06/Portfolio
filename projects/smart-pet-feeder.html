01 — THE BRIEF

View Scenarios PDF ↗
02 — TECHNICAL DEPTH & ACHIEVEMENTS
  • Firmware Optimization: Developed custom C++ firmware on an ESP32 utilizing the NimBLE-Arduino library to achieve a 50% reduction in RAM and Flash usage compared to standard Bluedroid.
  • Hardware-Software Co-Design: Integrated a mechanical dispensing mechanism with an HX711 amplifier and load cell. Implemented digital calibration routines to achieve automated, software-driven portion control within ±1% precision.
  • Event-Driven Communication: Built a GATT server using the Nordic UART Service (NUS) with event-driven callback handlers (onWrite, onConnect, onSubscribe) for low-latency command processing from a Kotlin-based Android app.
SKILLS USED: C++ Arduino (C++) EasyEDA PCB Assembly & Soldering
03 — OUTCOME

A working connected pet feeder prototype that feeds on schedule and confirms each portion by weight via automated digital calibration.
