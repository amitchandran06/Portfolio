01 — THE BRIEF

Integration of computer vision with high-level Simulink models and physical flight control units.

02 — TECHNICAL DEPTH & ACHIEVEMENTS
  • Real-time Computer Vision: Built a Python OpenCV pipeline on an edge Raspberry Pi to detect and track ArUco coordinates dynamically.
  • Networking & Control: Established a low-latency UDP link between Python and MATLAB/Simulink to pipe XYZ spatial telemetry.
  • Flight Controller Hardware: Interfaced high-level control loops with an Aero Selfie H743 flight controller to enable autonomous flight behaviors.
SKILLS USED: Python MATLAB Simulink
03 — OUTCOME

An ongoing development establishing real-time autonomous tracking and control between an edge computer vision device and physical flight hardware.
