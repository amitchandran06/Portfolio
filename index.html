Drone Vision & Control System

UCL Rover Team

End-to-end integration of a Raspberry Pi vision pipeline with MATLAB/Simulink and a physical flight controller.

  • Programmed an edge-computing vision pipeline in Python and OpenCV on a Raspberry Pi to track ArUco markers in real-time.
  • Established a robust UDP telemetry link to stream positional data seamlessly from the edge device to a MATLAB/Simulink model.
  • Interfaced the Simulink architecture directly with an Aero Selfie H743 flight controller to achieve an autonomous, end-to-end vision-to-flight pipeline.

Electronics Prototyping Station

Desktop Multimeter

A comprehensive 226-page NEA project integrating an Arduino MEGA, custom PCB, and 3D printed DFMA enclosure into an all-in-one testing station.

  • Engineered custom measurement circuitry (using ACS724 Hall-effect sensors and potential dividers) and routed the PCB in EasyEDA.
  • Programmed a 500+ line C++ firmware utilizing hardware interrupts (ISRs) and SPI communication for a 1.8-inch TFT display.
  • Designed a modular, multi-part PETG/PLA enclosure in Fusion 360, focusing heavily on Design for Manufacture and Assembly (DFMA).

Smart Pet Feeder

BLE IoT System

A complete IoT ecosystem featuring mechanical dispensing, custom load-cell circuitry, and ESP32 NimBLE firmware.

  • Developed bespoke C++ firmware for an ESP32 microcontroller utilising the NimBLE stack, achieving a 50% reduction in flash usage.
  • Engineered a mechanical dispensing mechanism fully integrated with custom load-cell circuitry and ESP32 firmware for automated portion control (±1%).
  • Implemented an event-driven GATT server architecture using the Nordic UART Service (NUS) enabling low latency communication with a custom Android client.
